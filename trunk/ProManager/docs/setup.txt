Required Softwares


JDK1.6 (Java Compiler required to compile and build java applications)
Maven2.8 (Build tool like ANT, Compiles java Application)
Netbeans6.5 (Development environment IDE)
Tortoise SVN or oneclicksubversion (Subversion Client)
Teamviewer (for remote desktop help)
